# Uncertainty Assessment of Social Media Situational Signals

This repository contains the code for the paper **"LLM-based Uncertainty Assessment of Social Media Situational Signals for Crisis Reporting"**

## Overview

The repository provides the scripts used in the framework:

- **Situational awareness classification**
- **Uncertainty assessment (plausibility and confidence)**
- **Uncertainty-aware crisis report generation**

These scripts illustrate the core components of the pipeline described in the paper.

## Repository Structure

classification/  
Scripts and prompts for situational awareness classification.

uncertainty_assessment/  
Scripts and prompts for plausibility and confidence estimation.

report_generation/  
Scripts and prompts for generating uncertainty-aware crisis reports.

## Notes

This repository provides reference implementations of the main components used in the study.  
It is intended for transparency and illustration rather than as a fully packaged pipeline.
